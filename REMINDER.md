## 📋 数据录入要求

### 必需字段（每个活动必须填写）
1. **标题** (title): 活动名称
2. **分类** (category): 瑜伽/冥想/户外探险/文化艺术/美食体验/节庆活动/其他
3. **时间** (time): 如 "08:30-09:45"
4. **日期** (weekdays): ["周一"] 或 ["周一", "周三"]
5. **地点** (location): 具体地址
6. **价格** (price): "免费" 或 "200฿"
7. **描述** (description): 详细介绍

### 数据文件位置
```
data/items.json
```

### 数据模板位置
```
data/activities-template.json (待创建)
```

