## 🔍 核查方法论

### 1. Google搜索策略

**搜索关键词模板：**
```
"[活动名称] Chiang Mai 2026"
"[英文名称] opening hours"
"[活动名称] 清迈 最新"
"[英文名称] Facebook"
```

**信息验证点：**
- ✓ 活动是否存在
- ✓ 是否仍在运营
- ✓ 基本信息是否准确
- ✓ 是否有官方网站

### 2. Facebook核查

**查找内容：**
- ✓ 官方页面最新帖子
- ✓ 营业时间声明
- ✓ 用户评价和问答
- ✓ 价格更新通知

**验证方法：**
```
facebook.com/[活动名称]
搜索 "[活动名称] Chiang Mai"
```

### 3. Instagram核查

**查找内容：**
- ✓ 官方账号
- ✓ 最新动态（Stories/Posts）
- ✓ 用户标记和评价
- ✓ 实际场景照片

**验证方法：**
```
instagram.com/[活动名称]
搜索标签 #[活动名称]ChiangMai
```

### 4. 数据更新标准

**需要更新的情况：**
- ❌ 营业时间错误
- ❌ 价格已变更
- ❌ 地址不准确
- ❌ 联系方式失效
- ❌ 活动已停止运营

**保持原样的情况：**
- ✓ 信息准确无误
- ✓ 只有微小差异（如价格几泰铢浮动）
- ✓ 信息在多个来源一致
