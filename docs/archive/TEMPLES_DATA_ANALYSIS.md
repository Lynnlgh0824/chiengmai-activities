## 📝 总结

### 🔴 必须修复的问题

1. **所有寺庙缺少"星期"字段** - 影响筛选和显示
2. **联系方式未添加到描述** - 用户无法获取联系方式
3. **松德寺数据不完整** - 缺少时间和费用

### 🟡 建议优化的字段

1. **添加"持续时间"** - 提供课程时长信息
2. **提取"最低价格"** - 便于价格筛选
3. **统一描述格式** - 包含所有关键信息（语言、联系方式等）

### ✅ 数据映射策略

**Excel 字段优先级：**
1. ✅ 活动标题 → 名称
2. ✅ 分类 → "冥想"
3. ✅ 地点 → 地点
4. ✅ 价格 → 价格 + 最低/最高价格
5. ✅ 时间 → 时间
6. 🔴 星期 → **必须补充**
7. 🔴 描述 → 描述（包含：适合人群、活动特点、语言、联系方式）
8. 🟡 持续时间 → **建议补充**
9. ✅ 需要预约 → 需要预约
