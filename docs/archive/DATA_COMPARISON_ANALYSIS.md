## 🔍 现有数据还可以优化的细节

### 1. 描述信息可以更详细

**当前状态：**
```json
"description": "需要自己带瑜伽垫，或15-20泰铢租。下雨天取消。"
"notes": "需自备瑜伽垫，或现场租用（15-20泰铢）。下雨天活动取消。建议提前到达。"
```

**问题：** 描述和notes内容重复

**建议优化：**
```json
"description": "位于古城西南角的免费社区瑜伽，环境优美，适合所有水平的练习者。"
"notes": "需自备瑜伽垫，或现场租用（15-20泰铢）。下雨天活动取消。建议提前到达。"
```

**优先级：** ⚠️ 中

---

### 2. minPrice/maxPrice字段可以更精确

**当前状态：**
```json
"price": "需购买咖啡"
"minPrice": 50,
"maxPrice": 100
```

**问题：** 部分活动的价格范围不够准确

**建议优化：**
- 为所有"免费"活动设置 minPrice: 0, maxPrice: 0
- 为所有"捐赠"活动设置合理的参考范围
- 为"需购买咖啡"活动添加更精确的价格说明

**优先级：** ⚠️ 中

---

### 3. timeInfo字段可以更规范

```json
"timeInfo": "固定频率活动"
```
