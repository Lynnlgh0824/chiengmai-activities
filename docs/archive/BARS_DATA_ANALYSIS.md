## 🎯 优化建议

### 1. 📅 补充"星期"字段（高优先级）

清吧通常每天都营业，建议填入：
```
星期: "周一,周二,周三,周四,周五,周六,周日"
```

### 2. 🪑 补充"需要预约"字段（高优先级）

部分清吧需要提前占位：
- **North Gate Jazz Co-Op**: 周末建议提前占位 → `"需要预约": "是"`
- **Good View Bar**: 河边景观位需提前占位 → `"需要预约": "是"`

### 3. 💰 提取价格范围（中优先级）

部分清吧有明确的人均消费：
- **Your Bar**: 人均300-500泰铢
  - `"最低价格": 300`
  - `"最高价格": 500`

### 4. 📝 补充描述字段（低优先级）

将特色描述补充到"描述"字段：
- 装修风格（工业风、木结构老房子）
- 氛围特点（氛围纯粹、适合年轻人）
- 景观优势（可俯瞰城区景色、河边景观）
- 预约提醒（周末座无虚席、需提前占位）
