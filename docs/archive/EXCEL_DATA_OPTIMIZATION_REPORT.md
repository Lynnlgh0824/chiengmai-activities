## 📋 新增字段说明

### `notes` 字段

**字段类型：** 文本（Text）
**是否必填：** 否
**用途：** 提供实用的注意事项和提示

**内容包括：**
- ⚠️ 重要提醒（如携带护照）
- ⏰ 时间建议（如售完即止、最佳访问时间）
- 👕 穿着建议（如运动鞋、瑜伽垫）
- 💰 支付方式（如建议携带现金）
- 🚫 注意事项（如闭馆时间、天气影响）
