## 🎯 统一格式规范

### 标准格式模板

**短描述（<50字）：**
```
[简短描述]

⚠️ 注意事项：
[关键信息]
```

**中等描述（50-150字）：**
```
[主要描述1-2句]

⚠️ 注意事项：
[补充说明]
```

**长描述（>150字）：**
```
[段落1：概述/特点]

[段落2：具体细节（语言、费用、时间等）]

⚠️ 注意事项：
[重要提醒/补充说明]
```

### 核心原则

1. **避免重复**：描述和注意事项不要重复相同内容
2. **补充信息**：注意事项应补充描述中未提及的关键信息
3. **⚠️符号**：全文只使用1次
4. **分段清晰**：长描述用 `\n\n` 分段，提高可读性
5. **标点规范**：中文使用全角标点（。！？，：），英文使用半角
