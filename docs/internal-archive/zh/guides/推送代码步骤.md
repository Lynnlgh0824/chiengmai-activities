## 方案 1: 使用 Personal Access Token（推荐）

### 步骤 1: 创建 GitHub Token

1. 访问: https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. 设置:
   - **Note**: `Chiang Mai Guide`
   - **Expiration**: 选择过期时间（建议90天）
   - **Scopes**: 勾选 `repo` (完整仓库访问权限)
4. 点击 **Generate token**
5. ⚠️ **重要**: 复制生成的 token（只显示一次！）

### 步骤 2: 使用 Token 推送

**方法 A: 直接推送（会提示输入密码）**
```bash
git push -u origin main
# 用户名输入: Lynnlgh0824
# 密码输入: 粘贴刚才的 token（不是GitHub密码）
```

**方法 B: 使用 Git Credential Helper**
```bash
# 设置使用 token（替换 YOUR_TOKEN）
git remote set-url origin https://YOUR_TOKEN@github.com/Lynnlgh0824/chiangmai-activities.git

# 推送
git push -u origin main
```

## 方案 2: 安装 GitHub CLI（最简单）

### 安装步骤

**macOS**:
```bash
brew install gh
```

**Windows**:
```bash
winget install --id GitHub.cli
```

### 认证并推送

```bash
# 1. 登录 GitHub
gh auth login

# 按照提示操作:
# - What account? -> GitHub.com
# - What protocol? -> HTTPS
# - Authenticate Git? -> Yes
# - How to authenticate? -> Login with a web browser

# 2. 推送代码
git push -u origin main
```

## 方案 3: 使用 SSH 密钥（长期使用推荐）

### 生成 SSH 密钥

```bash
# 1. 生成密钥
ssh-keygen -t ed25519 -C "your_email@example.com"

# 2. 查看公钥
cat ~/.ssh/id_ed25519.pub
```

### 添加到 GitHub

1. 访问: https://github.com/settings/keys
2. 点击 **New SSH key**
3. **Title**: `My MacBook`
4. **Key**: 粘贴刚才的公钥内容
5. 点击 **Add SSH key**

### 修改远程仓库为 SSH

```bash
git remote set-url origin git@github.com:Lynnlgh0824/chiangmai-activities.git
git push -u origin main
```

## ❓ 遇到问题？

### 错误 1: "Permission denied"
**原因**: Token 错误或过期
**解决**: 重新生成 Token

### 错误 2: "Repository not found"
**原因**: 仓库名称错误或未创建
**解决**: 确认在 GitHub 创建了 `chiangmai-activities` 仓库

### 错误 3: "SSL certificate problem"
**解决**: 执行 `git config --global http.sslVerify false`
