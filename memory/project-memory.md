# 项目总体记忆

> **项目名称**: Chiengmai Activities Platform (清迈活动策划管理系统)
> **最后更新**: 2026-02-25
> **维护者**: 项目团队

---

## 📋 项目概述

清迈活动策划管理系统，基于 Express.js 构建的活动管理平台。

### 核心功能
- 活动创建与管理
- 飞书表格集成
- 用户注册与登录

### 技术栈
- **运行时**: Node.js
- **框架**: Express.js 4.x
- **开发工具**: Nodemon

---

## 🎯 项目目标

1. 提供便捷的活动策划管理工具
2. 集成飞书协作平台
3. 支持多用户协作

---

## 💡 关键洞察

- 项目处于早期开发阶段
- 核心架构基于 Express.js
- 飞书集成是关键特性
